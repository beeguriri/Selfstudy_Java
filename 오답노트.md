### ⭐회의실배정, 요격시스템, ...
- 문제유형 : 정렬, 그리디
- 끝나는 시간 기준으로 오름차순 정렬
```java
Arrays.sort(targets, 
        (o1, o2) -> o1[1]==o2[1] ? o1[0]-o2[0] : o1[1]-o2[1]))
        
int last = target[0][1];
int count = 1;

for(int [] target : targets) {
    if(target[0]>=last) {
        count++;
        last = target[1];
    }    
}
```

- 선택에 따른 값의 계산이 필요할 때는 dp로 풀기
```java
for(int i=1; i<=n; i++) {
    //상담원 배정이 가능할때
    if(i + t[i] <= target) {
        d[i+t[i]] = Math.max(d[i+t[i]], d[i] + p[i]);    
    }
    //상담원 배정이 불가능 할 때
    d[i+1] = Math.max(d[i], d[i+1]);
}
```
